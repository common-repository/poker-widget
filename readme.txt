== Description ==

Poker Tournaments Finder: Displays Live poker / texas holdem tournaments from all the leading poker rooms with full information and designed layout.
Casino Jackpots Finder: Displays Live Casino Jackpots from all the leading casinos with full information and designed layout.



== Installation ==

This section describes how to install the plugin and get it working.



   1. choose the right widget version for your wordpress version.
   2. Upload poker-widget.php to the /wp-content/plugins/ directory
   3. Activate the plugin through the 'Plugins' menu in WordPress
   4. Go to the widget menu and configure the poker widget.
